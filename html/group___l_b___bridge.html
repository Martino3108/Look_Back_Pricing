Creates a LookBack pricer instance (ASCII dates, portable ABI).

Parses value/maturity dates formatted as dd-mm-yyyy, builds a C++ look_back object, and returns it as an opaque handle.

Error model:

  • Returns nullptr on failure and sets the thread-local last error.
  • Retrieve the error with LB_GetLastErrorA().
Parameters
S0Spot price at valuation.
value_date_dd_mm_yyyyValuation date string "dd-mm-yyyy".
maturity_date_dd_mm_yyyyMaturity date string "dd-mm-yyyy".
sigmaVolatility (annualized, numeric).
interest_rateRisk-free rate (annualized, numeric).
option_asciiOption type as ASCII (e.g. 'c' for call or 'p' for put).
hFinite-difference step used by Greeks.
day_count_convDay-count convention (LB_DayCountConv).
Returns
LB_Handle Opaque handle, or nullptr on error.
