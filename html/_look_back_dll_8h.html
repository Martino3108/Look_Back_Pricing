Detailed Description

C ABI wrapper for the LookBack pricer (Excel/VBA-friendly).

This header exposes a stable C interface to the underlying C++ implementation of a lookback option pricer. The API uses an opaque handle (LB_Handle) to manage object lifetime across language boundaries (e.g., VBA).

Key design points:

  • No C++ exceptions cross the ABI boundary: functions catch exceptions and store an error message in a thread-local buffer retrievable via LB_GetLastErrorA().
  • Platform calling conventions: LB_CALL and LB_API handle Windows vs macOS/Linux.
  • Day count conventions: the C enum maps to the C++ DayCountConv.
