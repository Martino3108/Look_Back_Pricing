\doxysubsection{Detailed Description}
C ABI wrapper for the Look\+Back pricer (Excel/\+\+VBA-\/friendly). 

This header exposes a stable C interface to the underlying C++ implementation of a lookback option pricer. The API uses an opaque handle ({\ttfamily \doxylink{group___l_b___bridge_ga11b9592738d656b17b24cc3a9f42e9d8}{LB\+\_\+\+Handle}}) to manage object lifetime across language boundaries (e.\+g., VBA).

Key design points:\+
\begin{DoxyItemize}
\item {\bfseries{No C++ exceptions cross the ABI boundary}}:\+ functions catch exceptions and store an error message in a thread-\/local buffer retrievable via \doxylink{group___l_b___bridge_gaf98edfebf3ee03b2efdf7003c51db016}{LB\+\_\+\+Get\+Last\+Error\+A()}.
\item {\bfseries{Platform calling conventions}}:\+ LB\+\_\+\+CALL and LB\+\_\+\+API handle Windows vs mac\+OS/\+\+Linux.
\item {\bfseries{Day count conventions}}:\+ the C enum maps to the C++ {\ttfamily \doxylink{_date___dealing_8h_aa00d31f388138ee974de37f288651cd0}{Day\+Count\+Conv}}.
\end{DoxyItemize}
