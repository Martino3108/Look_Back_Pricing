\doxysubsection{Detailed Description}
Core C++ lookback option pricer interface. 

\begin{DoxyAuthor}{Author}
Vincenzo Martino Pio Arena Marco De Luca
\end{DoxyAuthor}
Defines the {\ttfamily \doxylink{classlook__back}{look\+\_\+back}} class which encapsulates contract inputs (dates, option type), market parameters (sigma, rate), and numerical controls for Greeks.

The class offers:\+
\begin{DoxyItemize}
\item Monte Carlo pricing (option payoff estimated under GBM assumptions).
\item Greeks computed via finite differences around the stored baseline parameters.
\item Simple graph helpers for price/\+delta as a function of spot.
\end{DoxyItemize}

Error model:\+
\begin{DoxyItemize}
\item The core C++ implementation may throw exceptions (e.\+g., \doxylink{class_invalid___parameters}{Invalid\+\_\+\+Parameters}).
\item The ABI bridge (\doxylink{_look_back_dll_8cpp}{Look\+Back\+Dll.\+cpp}) is responsible for catching and reporting them. 
\end{DoxyItemize}
