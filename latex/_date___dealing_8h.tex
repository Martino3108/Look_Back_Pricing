\doxysubsection{Detailed Description}
\doxylink{struct_date}{Date} parsing and year-\/fraction computation (day-\/count conventions). 

This module provides:\+
\begin{DoxyItemize}
\item Parsing of date strings in format "{}dd-\/mm-\/yyyy"{} into {\ttfamily std::\+chrono::\+year\+\_\+month\+\_\+day}.
\item Basic date difference in days.
\item Year-\/fraction computation under standard day-\/count conventions.
\end{DoxyItemize}

Exceptions:\+
\begin{DoxyItemize}
\item Parsing throws {\ttfamily std::\+invalid\+\_\+argument} on invalid date format.
\item Unknown conventions throw {\ttfamily std::\+invalid\+\_\+argument}.
\end{DoxyItemize}
