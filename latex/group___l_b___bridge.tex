Creates a Look\+Back pricer instance (ASCII dates, portable ABI). 

Parses value/\+maturity dates formatted as {\bfseries{dd-\/mm-\/yyyy}}, builds a C++ {\ttfamily \doxylink{classlook__back}{look\+\_\+back}} object, and returns it as an opaque handle.

Error model:\+
\begin{DoxyItemize}
\item Returns {\ttfamily nullptr} on failure and sets the thread-\/local last error.
\item Retrieve the error with \doxylink{group___l_b___bridge_gaf98edfebf3ee03b2efdf7003c51db016}{LB\+\_\+\+Get\+Last\+Error\+A()}.
\end{DoxyItemize}
