Detailed Description

Implementation of Monte Carlo pricing and finite-difference Greeks for lookback options.

The pricing method uses OpenMP to parallelize Monte Carlo draws. Each thread uses its own RNG instance seeded deterministically from the thread id to prevent data races.

Numerical notes:

  • Antithetic variates are used (plus/minus Z).
  • Uniform draws are clamped to avoid log(0).
Warning
Some Greeks infer the Monte Carlo sample size as N = 1/h^4, which can become huge.
