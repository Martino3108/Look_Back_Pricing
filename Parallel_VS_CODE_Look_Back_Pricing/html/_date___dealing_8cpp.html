Detailed Description

Implementation of date parsing and year-fraction computations.

Implements 30/360 EU, 30/360 US, and ACT/ACT ISDA conventions using std::chrono. The dispatcher yearFraction() selects the appropriate convention and returns the fraction of years between two dates.
